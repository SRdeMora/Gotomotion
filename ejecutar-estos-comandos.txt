# Ejecuta estos comandos después de crear el repositorio en GitHub

# Reemplaza NOMBRE_DEL_REPO con el nombre que pusiste en GitHub
# Por ejemplo, si se llama "go2motion-awards", sería:
# git remote add origin https://github.com/SRdeMora/go2motion-awards.git

# Verificar que se agregó correctamente
git remote -v
